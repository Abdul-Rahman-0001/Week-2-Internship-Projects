# Week 2 Internship Projects

## Projects Included
1. Windows Hardening
2. Firewall Configuration
3. Password Policy Setup

## 1. Windows Hardening

### Objective
Improve Windows system security by applying hardening techniques.

### Tasks Performed
- Enabled Windows Firewall  
- Applied recommended security settings  
- Disabled unnecessary services  

### Tools Used
- Windows Security Settings  
- Local Security Policy  

### Result
System configuration strengthened, attack surface reduced.

## 2. Firewall Configuration

### Objective
Configure advanced firewall rules for inbound and outbound traffic.

### Tasks Performed
- Created basic inbound & outbound rules  
- Applied advanced firewall settings  
- Tested rule effectiveness  

### Tools Used
- Windows Firewall / UFW  

### Result
Unauthorized access blocked, network traffic controlled.

## 3. Password Policy Setup

### Objective
Apply strong password policies to improve user account security.

### Tasks Performed
- Set minimum password length  
- Enabled complexity requirements  
- Configured maximum password age  

### Tools Used
- Local Security Policy  

### Result
Enhanced account security, reduced risk of weak passwords.
